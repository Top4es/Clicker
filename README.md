# 🎮 Крипто Кликер
Полноценная игра-кликер на PyQt6 с прогрессивной механикой, улучшениями, мини-играми и системой уровней.

## 📋 Описание
Это классический кликер с тематикой криптовалют. Задача игрока - кликать по кнопке, зарабатывать монеты, покупать улучшения и прокачиваться до максимального уровня.

---

## 🚀 Запуск игры
```bash
cd Clicker
python main.py
```

### Требования:
- Python 3.10+
- PyQt6
```bash
pip install PyQt6 PyQt6-svg
```

## 🛒 Магазин улучшений
В игре **3 категории улучшений**, в каждой по 10 улучшений:

### ✨ Пассивный доход
Улучшения увеличивают количество монет, начисляемых каждую секунду.
- 0 уровень: базовый эффект
- 1 уровень: ×2 эффект
- 2 уровень: ×3 эффект
- и так далее...

### 👆 Сила клика
Увеличивают количество монет за одно нажатие кнопки.
- Эффект растет в 2 раза за каждый уровень улучшения

### 📈 Прокачка уровня
Увеличивают скорость получения прогресса к следующему уровню.
- Минимальный эффект всегда минимум 1

💡 **Важно**: стоимость каждого следующего уровня увеличивается в 2 раза.

## 🎲 Мини-игры и ивенты

### 🔄 Задание смены кнопки
Каждую секунду 0.5% шанс появиться задание:
- Игрок обязан нажимать только указанную кнопку: левая, правая или пробел
- Бездействие больше 5 секунд = полный сброс всего прогресса!
- Длительность 60 секунд
- После прохождения задания следующее будет на другую кнопку

### 🎁 Плавающая кнопка бонуса
Случайно появляется на экране на 4 секунды. При нажатии дает один из бонусов:
- ✅ Двойной доход за клик на 10-30 секунд
- ✅ Двойной прогресс уровня на 10-30 секунд
- ✅ Тройной пассивный доход на 10-30 секунд
- ✅ Случайное количество монет от 50 до 500

### ⚡ Задание на клики
Необходимо накликать 20 раз за 5 секунд. Награда: тройной доход за клик 15-45 секунд.

### 💰 Задание на заработок
Необходимо заработать указанное количество монет за N секунд. Количество монет адаптируется под текущий прогресс игрока. Награда: четверной пассивный доход 15-45 секунд.

## 💾 Сохранение
Прогресс автоматически сохраняется в файл `save.json` в формате JSON. При запуске игры прогресс автоматически загружается.

Сохраняются:
- Баланс
- Общее количество кликов
- Все игровые показатели
- Уровни всех улучшений
