# 🎮 Игра «Крипто Кликер»

## 📝 Общее описание
Игра-кликер с прогрессивной системой уровней, магазином улучшений, случайными мини-играми и временными бонусами. Разработана на Python с использованием фреймворка PyQt6.

---

## 📚 Используемые библиотеки

| Библиотека | Назначение |
|------------|------------|
| `PyQt6` | Основной графический фреймворк для создания интерфейса |
| `sys` | Работа с системными параметрами и завершение программы |
| `os` | Работа с файловой системой, пути к файлам |
| `json` | Сохранение и загрузка прогресса игры в формате JSON |
| `random` | Генерация случайных событий, появление мини-игр |

---

## 🏗️ Структура программы

### 1. Класс `CryptoClickerLogic`
Класс логики игры, содержит все игровые данные и вычисления:

| Метод | Описание |
|-------|----------|
| `get_current_level()` | Определяет текущий уровень по общему количеству кликов |
| `get_level_progress()` | Рассчитывает прогресс в процентах до следующего уровня |
| `process_click()` | Обрабатывает клик по основной кнопке, начисляет баланс и прогресс |
| `process_passive_income()` | Добавляет пассивный доход к балансу каждую секунду |
| `get_upgrade_cost()` | Рассчитывает стоимость следующего уровня улучшения |
| `buy_upgrade()` | Обрабатывает покупку улучшения, списывает валюту и применяет бонус |

---

### 2. Класс `ShopWindow`
Окно магазина улучшений:

| Метод | Описание |
|-------|----------|
| `init_ui()` | Создает интерфейс магазина с вкладками |
| `update_theme()` | Обновляет стилизацию магазина согласно текущему уровню |
| `create_upgrade_row()` | Создает строку с отдельным улучшением |
| `update_shop()` | Обновляет все улучшения и их стоимость |

---

### 3. Класс `FloatingBonusButton`
Плавающая кнопка случайного бонуса, появляющаяся на экране. Содержит анимацию появления и таймер самоликвидации через 4 секунды.

---

### 4. Класс `MainWindow`
Главное окно игры:

| Метод | Описание |
|-------|----------|
| `init_ui()` | Создает весь интерфейс главного окна |
| `paintEvent()` | Рисует градиентный фон окна |
| `check_minigame_event()` | Проверяет возможность появления мини-игры каждую секунду |
| `collect_bonus()` | Обрабатывает получение бонуса с плавающей кнопки |
| `minigame_tick()` | Обновляет таймер и прогресс активной мини-игры |
| `update_interface()` | Обновляет все элементы интерфейса с актуальными значениями |
| `show_result()` | Показывает результат мини-игры под кнопкой клика |
| `save_progress()` | Сохраняет весь прогресс игры в файл `save.json` |
| `load_progress()` | Загружает сохраненный прогресс при запуске игры |
| `closeEvent()` | Показывает подтверждение при попытке выхода из игры |

## ⚙️ Основные механики

### ✅ Система уровней
Всего 5 уровней с уникальными стилями и названиями криптовалют:
1. **Dodgecoin** - 0 → 5000 кликов
2. **Litecoin** - 5000 → 14000 кликов
3. **Solana** - 14000 → 26000 кликов
4. **Etherium** - 26000 → 50000 кликов
5. **Bitcoin** - 50000 → бесконечность

### ✅ Магазин улучшений
3 категории улучшений по 6 штук в каждой:
1. **Пассивный доход** - увеличивает доход каждую секунду
2. **Сила клика** - увеличивает доход за один клик
3. **Прокачка уровня** - ускоряет получение прогресса уровней

Стоимость каждого следующего уровня улучшения возрастает в 2 раза.

### ✅ Случайные мини-игры
Каждую секунду с 3% шансом появляется одна из трех мини-игр:
1. 🎁 Плавающая бонусная кнопка
2. ⚡ Задание накликать определенное количество раз за время
3. 💰 Задание заработать определенное количество монет за время

### ✅ Временные бонусы
Получаемые за прохождение мини-игр бонусы складываются и могут действовать одновременно:
- Множитель дохода за клик
- Множитель прогресса уровня
- Множитель пассивного дохода
- Прямая награда монетами

## 💾 Сохранение прогресса
Весь прогресс автоматически сохраняется при выходе в файл `save.json`, при следующем запуске игры загружается автоматически.
